A simple user friendly plugin which converts the Twenty Fourteen theme to full width

== Description ==

Are you using the Twenty Fourteen theme and like its design, but you don't like that theme is limited to 1260px of width? This plugin provides you with a user-friendly way to add full width to your Twenty Fourteen theme. 

No coding required, plug and play, lightweight solution.

== Installation ==

Installing "Full Width for Twenty Fourteen" can be done either by searching for "Full Width for Twenty Fourteen" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Download the plugin via WordPress.org
2. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Enjoy the new full-width design of the Twenty Fourteen theme

== Frequently Asked Questions ==

= Does this plugin affect theme header width as well =

Yes, both the content and the site header are stretched to full width

= Are child themes based on Twenty Fourteen parent theme also supported =

Most likely yes, depends on the level of customization. Heaviliy customized child themes might override the default theme structure.

= Can I use this plugin with any other theme? =

Short answer: no. Long answer: If tbeme uses the same classess as the Twenty Fourteen theme, then it might work.
